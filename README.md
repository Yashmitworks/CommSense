CommSense – AI Communication Coach

CommSense is an intelligent communication analysis platform designed to enhance high-stakes communication performance through real-time behavioral assessment. The system combines computer vision, emotion recognition, gaze tracking, speech analysis, and large language models to detect inconsistencies between verbal communication and non-verbal cues.

Using facial emotion recognition, eye-contact tracking, speech transcription, and contradiction detection algorithms, CommSense identifies situations where a speaker's facial expressions conflict with their spoken words. The platform then generates AI-powered coaching recommendations to improve confidence, credibility, trustworthiness, and overall communication effectiveness.

Key Features
Real-time facial emotion detection using MediaPipe and DeepFace
Eye-contact and gaze-tracking analysis
Speech-to-text transcription using Google STT or Whisper
Verbal–facial contradiction detection
Confidence, trust, nervousness, and engagement scoring
AI-generated communication coaching and feedback
Interactive dashboard with live performance metrics
Session recording and report generation
Offline operation support (except AI coaching services)

Objective

The goal of CommSense is to bridge the gap between what people say and how they unconsciously communicate, enabling users to identify behavioral inconsistencies and improve communication effectiveness through AI-driven feedback.
